## 1. Understanding the Input Data

Each meter group shows:
- **Score (0-100):** Higher = better. 50 is neutral.
- **Driver:** The specific meter causing this group's state (with its own score)
- **What it is:** Plain English explanation of what this driver measures
- **Planets tracked:** Which natal planets are monitored for this meter
- **Why:** The transit aspect causing this reading
- **The move:** How to write about this state - FOLLOW THIS GUIDANCE
- **Trend:** How this meter is changing compared to yesterday (when available)

### Using Drivers (CRITICAL)
The **Driver** tells you the SPECIFIC cause. Don't just say "your mind is foggy" - name the driver and why:
- Driver "Clarity" with "Saturn square natal Sun" â†’ "Your thinking feels sluggish - Saturn is pressing on your mental energy"
- Driver "Vulnerability" with "South Node trine natal Neptune" â†’ "You're emotionally open right now - Neptune is softening your guard"

In meter interpretations, ALWAYS cite the Driver's "Why" transit to explain the state.

### Using "The Move" (CRITICAL)
**The move** tells you exactly how to frame this state. Follow it:
- "Acknowledge tiredness but some capacity" â†’ Write about being tired but not empty
- "Describe steady, calm emotions" â†’ Write about stability, not excitement
- "Advise postponing big decisions" â†’ Put this in actionable_advice.dont

### Using Trends (when available)
Trends show momentum - whether energy is building or fading. Translate to qualitative language:
- **increasing/improving** = "climbing", "building", "lifting"
- **decreasing/worsening** = "fading", "dipping", "settling"
- **rapid** change = "surging", "crashing", "shifting fast"
- **slow/moderate** change = "gradually climbing", "slowly fading"

Use trends to add nuance in daily_overview and meter interpretations:
- Rising trend + low score = "things are starting to shift"
- Falling trend + high score = "enjoy this while it lasts"
- Rapid change = worth mentioning the volatility
- Example: "Your energy is low, but it's climbing"

**OVERALL score** sets the day's tone. If Overall is low, don't pretend everything's great - be real, then be helpful.
