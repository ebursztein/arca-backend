## 1. Understanding the Input Data

Each meter group shows:
- **Score (0-100):** Higher = better. 50 is neutral.
- **Driver:** The specific meter causing this group's state (with its own score)
- **What it is:** Plain English explanation of what this driver measures
- **Planets tracked:** Which natal planets are monitored for this meter
- **Why:** The transit aspect causing this reading
- **The move:** How to write about this state - FOLLOW THIS GUIDANCE
- **Trend:** How this meter is changing compared to yesterday (when available)

### The 4 Quality Buckets
Scores map to 4 distinct states. Know where you are:

| Score Range | Quality | What It Means | How to Write About It |
|-------------|---------|---------------|----------------------|
| 0-25 | **Challenging** | Actively difficult. Real friction. | Be direct about the struggle. Name what's hard. Then give a concrete way through. |
| 25-50 | **Turbulent** | Unsteady. Not terrible, but not reliable. | Acknowledge the instability. "Choppy waters" energy. Advise caution, not avoidance. |
| 50-75 | **Peaceful** | Stable. Nothing dramatic. | This is neutral-good. Don't oversell it. "Steady" is the vibe. |
| 75-100 | **Flowing** | Active support. Things click. | This is where things work. Encourage action. Lean into it. |

**Important:** Don't treat 50 as "bad" - it's neutral.

### Using Drivers (CRITICAL)
